Blockchain

This project was part of a Hackathon organised by "GlobalShala" and hosted by "Hackerearth", in which we created a prototype under the theme "Blockchain". We qualified the idea submission phase with our idea "Implementing Blockchain principles to issue and verify Badges in organisations". We built two consecutive models one of which was built on "remix.ethereum" using "Solidity" for building Smart Contracts, and showing how the smart contract is deployed on each of he nodes in network and how they spend ethereum for issuing and verification process.
And the second part was built using python showing the backend infrastructre of the blockchain and how the nodes on network communicate and keep the data in sync and prevent any kind of forgery.
